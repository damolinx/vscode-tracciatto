## Logs

Traciatto writes diagnostic information to the **Traciatto** output channel.
You can adjust the log level using **Developer: Set Log Level** and selecting **Traciatto**.
See [documentation](https://code.visualstudio.com/updates/v1_73#_setting-log-level-per-output-channel) for details.
